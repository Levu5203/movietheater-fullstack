# Run commands

## Run application with dotnet cli

1. Restore the solution

    ```bash
    dotnet restore
    ```

2. Build the solution

    ```bash
    dotnet build
    ```

3. Run MovieTheater.API

    ```bash
    dotnet run --project MovieTheater.API
    ```

4. Dotnet Watch
    
    ```bash
    dotnet watch --project MovieTheater.WebAPI
    ```
